# Catalysts Skies App

Catalysts Skies App is a client application for the Catalysts Skies economy simulation platform. It was designed to provide an interface for managing virtual airline companies, tracking flights, and interacting with a persistent aviation economy connected to flight simulator data.

This project is no longer actively developed and is preserved as an abandoned client prototype.

---

## Overview

The application served as a frontend client for the Catalysts Skies backend system, enabling interaction with a simulated airline economy driven by flight simulator activity.

It was intended to support:

- Virtual airline company management
- Flight scheduling and tracking integration
- Economic dashboards and analytics
- Multiplayer airline economy participation
- Integration with flight simulators such as Microsoft Flight Simulator and X-Plane

---

## Features

- User authentication and company profiles
- Flight planning and tracking interface
- Economic statistics and business performance dashboards
- Notification and messaging system (partial/experimental)
- Flight simulator integration layer (conceptual/partial)
- Responsive UI design for desktop and mobile targets

---

## Status

This project is abandoned.

Development was deferred in favor of backend-focused work on `CatalystsSkies.Service`. The client application was not completed and did not reach production use.

The repository is preserved for reference and experimentation.

---

## Notes

This project was intended as the primary user-facing interface for the Catalysts Skies ecosystem but was never fully integrated with a stable production backend or live simulator pipeline.
